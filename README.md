# Todo Manager

## It is a MERN Stack application. 

### Funconality
- User can add todo with title and description
- User can edit and update todo
- User can move todo to completed section
- User can see list of todo

#### Technologies Used
- React.js
- React Hook Form
- React DOM
- React Toastify
- Tailwind CSS
- DaisyUI
- Heroku
- Node.js
- MongoDB
- Express 
- Cors
- dotenv
- Vercel
